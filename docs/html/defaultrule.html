Default Rule

Consider a predicate

  • 'sweep' P ( ANY )
i.e. with one parameter. For a given functor
  • f ( T1 , T2 , ... , Tn )
the default rule has the form
  • 'rule' P( f(X1 , X2 , ... , Xn) ) :
    P( X1 ) P( X2 ) ... P( Xn )
where a member P(Xi) is omitted if the corresponding type Ti is not defined by terms (e.g if it is INT).
